## Run Locally (NOT FINAL)

Clone the project

```bash
  git clone https://gitlab.fdmci.hva.nl/lag/lag2324/lag_gritla.git
```

Go to the project directory

```bash
  cd lag_gritla
```

Navigate to assignment

```bash
 cd week_1/1.1
```

Compile the java file
```bash
  javac example
```

Run the compiled java file
```bash
  java example
```
